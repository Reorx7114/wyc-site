# 圖片命名說明

請將正式圖片放在此資料夾，並維持以下檔名：

- `wang-portrait.png`：首頁左側王永才真人去背照
- `chibi-main.png`：首頁右側主要 Q 版人物
- `chibi-blue.png`：其他區塊使用的 Q 版人物
- `vest-reference.png`：背心色彩與造型參考
- `hero-bg.png`：首頁主視覺背景（可選）
- `chongyang-bridge.png`：首頁重陽大橋手繪插畫
- `line-qrcode.jpg`：LINE 官方帳號 QR Code
- `facebook-qrcode.png`：Facebook QR Code
- `wang-sticker-main.png`：首頁真人照片與 Q 版娃娃合成主視覺
- `wang-sticker-cropped.png`：去除尺寸標線後的首頁正式主視覺
- `pending-main-reference.png`：使用者提供但尚未正式套用的主視覺參考圖
- `wang-real-person-crop.png`：從參考圖裁出的真人人物
- `wang-chibi-person-crop.png`：從參考圖裁出的 Q 版人物

未放入正式圖片前，網站會顯示設計好的替代視覺，不會出現破圖。
