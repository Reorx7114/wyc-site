# WYC Site V1.4 交付摘要

## 功能變更

- 服務申請成功後產生不可預測的公開案件編號，格式為 `SR-XXXXXX`。
- 送件成功畫面顯示案件編號與案件查詢入口。
- 新增 `/track`，可使用案件編號與電話末四碼查詢案件進度。
- 新增 `/track/recover`，可使用姓名與完整電話找回案件編號。
- 後台案件管理新增公開回覆，供里民於案件查詢頁查看。
- 案件狀態變更會立即寫入資料庫，並同步更新狀態快速篩選數量。
- 查詢與找回 API 使用統一查無資料訊息及 Rate Limit。
- Header 桌機與手機主選單新增「案件查詢」入口。
- 移除聯絡頁已停用的 Google Form 預留區。

## 資料表異動

`service_requests` 採向後相容方式新增：

- `public_reference`：對外顯示的唯一亂數案件編號。
- `public_reply`：供案件查詢頁顯示的公開回覆。

既有案件由 migration 自動回填公開案件編號。資料庫內部仍使用原有 UUID 主鍵，既有 Blog、Event、Video 與圖片資料不受影響。

## Migration

Migration 檔案：

- `docs/v1.4-case-tracking-migration.sql`

執行內容：

- 向後相容新增 V1.4 欄位。
- 建立唯一案件編號產生與回填機制。
- 保持 `service_requests` RLS 啟用。
- 不開放前台直接讀取案件資料。
